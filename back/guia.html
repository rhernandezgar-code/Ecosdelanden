Guía de Inicio

Pasos para clonar y configurar el template

Ver Repositorio

Crear Proyecto

  1. Clonar desde template - Haz clic en Use this template en GitHub
  2. Crear nuevo repositorio - Dale un nombre a tu proyecto
  3. Clonar localmente - Ejecuta en tu terminal:
git clone https://github.com/tu-usuario/tu-nuevo-proyecto.git
cd tu-nuevo-proyecto

Configuración Mínima

Archivos a Modificar

  • _config.yml - Cambia título y URL
  • css/style.css - Ajusta colores y estilos
  • index.html - Personaliza el contenido

Cómo Funciona

Estructura del Template

  • _layouts/ - Plantillas HTML base
  • _includes/ - Componentes reutilizables
  • css/ - Estilos con sistema skeleton
  • img/ - Recursos e imágenes

Flujo de Trabajo

  1. Edita las páginas HTML en la raíz
  2. Personaliza los layouts en _layouts/
  3. Modifica los estilos en css/style.css
  4. Sube los cambios a GitHub
  5. GitHub Pages genera automáticamente el sitio
